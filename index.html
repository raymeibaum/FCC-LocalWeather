Random Quote Generator - Click for Details

  • Description

    Random Quote Generator is an intermediate front end project on Free Code Camp. The program allows you to click to view a new quote, or to tweet the quote you're currently viewing.

  • Features

    The random quote generator makes an API call to retrieve an array of random quotes. It also features a Tweet button that will tweet the current quote the user is viewing

  • Challenges

    The biggest challenge for me on this project was figuring out the API call. Unfortunately, because the quote API endpoint is served over http, I am forced to also serve this page over http. I wish that I could find a way to make HTTPS work with it.

    My other challenge was figuring out how to tweet long strings that feature string-unfriendly characters. The answer: encodeURIComponent()

  • Technology

    jQuery, Bootstrap, AJAX, Programming Quotes API by Adam Pope, Tweet button API
