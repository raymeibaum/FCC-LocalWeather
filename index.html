Local Weather - Click for Details

  • Description

    Local Weather is an intermediate front end project on freecodecamp.com. The program allows you to get relevant weather information based on your current location.

  • Features

    Local Weather accesses your devices current location and then makes API calls to Google for geolocation information as well as Forecast.io for the weather forecast.

  • Challenges

    Working with location was easier than expected. Honestly, the hardest part was formatting the results correctly. I used jQuery more in this app than ever before.

  • Technology

    jQuery, Bootstrap, AJAX, Forecast.io API, Google Geolocation API
